# travel-server

The travel project server side

## Clone

```bash
git clone https://github.com/yuusheng/travel-server.git
```

## Project setup

```bash
yarn
```

### Compiles and hot-reloads for development

```bash
yarn serve
```

- koa-body：解析请求体
- bcryptjs：密码加密
- gravatar：全球公认头像
- jsonwebtoken：jwt token
