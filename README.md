# travel-server

The travel project server side

## Clone

```bash
git clone https://github.com/yuusheng/travel-server.git
```

## Project setup

```bash
yarn
```

### Compiles and hot-reloads for development

```bash
yarn serve
```

dependencies:

```json
"dependencies": {
  "bcryptjs": "^2.4.3",
  "gravatar": "^1.8.2",
  "jsonwebtoken": "^8.5.1",
  "koa": "^2.13.4",
  "koa-body": "^4.3.0",
  "koa-router": "^10.1.1",
  "mongoose": "^6.2.6"
}
```

- koa-body：解析请求体
- bcryptjs：密码加密
- gravatar：全球公认头像
- jsonwebtoken：jwt token
